# video-chat-app

A simple multi-user video chat app with WebRTC.  

Functionality:

-	Pause Video 
-	Mute Audio
-	Share Room ID for invites 

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
